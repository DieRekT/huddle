# Intro Slide Images

This directory contains screenshot images for the intro slides.

## Required Images

1. **slide-1-create-room.png** - Screenshot of the join/create room screen showing:
   - Name input field
   - Viewer/Mic role selector
   - Create Room button

2. **slide-2-viewer-dashboard.png** - Screenshot of the viewer dashboard showing:
   - Topic header
   - Summary card
   - Key points list
   - Live transcript

3. **slide-3-qr-invite.png** - Screenshot of the QR code invite modal showing:
   - QR code
   - Room code
   - Share options

4. **slide-4-mic-screen.png** - Screenshot of the mic screen showing:
   - Large mic icon
   - Room code
   - Start Mic button

5. **slide-5-full-room.png** - Screenshot of the "Read the Room" full overview showing:
   - Complete summary
   - All key points
   - Room insights

## Image Requirements

- Format: PNG (with transparency preferred)
- Recommended size: 1200x800px (3:2 aspect ratio)
- Style: Actual app screenshots (the CSS will add cartoonized effects)
- Quality: High resolution for crisp display on all devices

## How to Create

1. Take screenshots of your actual app screens
2. Optionally enhance/edit them (add annotations, highlights)
3. Save as PNG files with the exact names listed above
4. Place in this directory

The CSS will automatically apply cartoonized effects (enhanced saturation, contrast, shadows, animations).
